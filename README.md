### ⚙️ Installation

**Build from source:**

1. Clone the OOPs-Project repository:
```sh
❯ git clone https://github.com/darshit001/OOPs-Project
```

2. Navigate to the project directory:
```sh
❯ cd OOPs-Project
```

3. Run Project:
```sh
❯ python run main.py
```
